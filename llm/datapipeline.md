# 简介
Data Pipeline 数据平台是 AI 系统的燃料工厂和质量控制中心. 它不直接做决策和控制, 而是决定了模型能否学习到真实、安全、高效的行为.
Data Pipeline 数据平台将海量、原始、异构的传感器数据, 转化为高质量、结构化、可训练的机器学习样本.

## 核心功能模块
### 数据采集
将智能设备, 包括自动驾驶汽车的摄像头、激光雷达、毫米波雷达、GPS/IMU、CAN 总线、麦克风等数据, 机器人的 RGB—D 相机、关节编码器、力传感器、语音指令、触觉反馈等.

### 数据传输与存储
采用 proto、parquet、ros bag 等高效压缩格式减少带宽传输成本. 采用 HDFS、S3、Minio 等管理 PB 级数据.

### 数据预处理
比如对齐不同传感器的时间戳, 校准相机-激光雷达标定参数, 剔除无效数据如遮挡、偏色、设备故障、极端天气等, 脱敏模糊人脸、车牌等.

### 数据标注
通过自动化标注、人工标注等方式对数据进行 2D/3D 画框, 对关键场景包括事故、corner cases 由专业标注团队复核.
