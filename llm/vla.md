# 简介
VLA(Video-Language-Action, 视觉-语言-动作)是一种前沿的多模态人工智能模型, 它将视觉感知、语言理解和物理动作统一在一个端到端的框架中.

## openvla
是一个开源的基于 7b 参数的视觉-语言-动作模型, 旨在通过推动通用具身智能机器人操控技术的研究.

## vla-0
是一种高性能的 vla 基础模型.

## smolvla
是 hugging face 推出的一个轻量级、开源的视觉-语言-动作模型, 面向机器人操控领域. 专门为低成本机器人硬件设计, 旨在让机器人通过理解视觉图像和文本指令来生成机械臂动作. 它仅有 450M 参数量, 可以在单个消费级 GPU 上进行训练和推理.
