# 简介
目标追踪相关.

## 目标追踪
通过 dlib 提供的算法库, 可以训练目标追踪模型. 然后从视频流中对目标进行追踪. 已经训练好的 demo 可以参考 [github - dlib-object-tracking](https://github.com/LaggyHammer/dlib-object-tracking).
